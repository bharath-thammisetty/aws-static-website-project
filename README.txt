AWS Static Website Hosting Project

Student Name: Bharath Thammisetty

Project Description:
This project demonstrates how to host a static website using Amazon S3 and distribute the website globally using Amazon CloudFront.

The website files were uploaded to an Amazon S3 bucket and the bucket was configured to support static website hosting. Public access permissions were added through a bucket policy so that the website is accessible to anyone on the internet.

Amazon CloudFront was configured to retrieve the website files from the S3 bucket and distribute them through the CloudFront Content Delivery Network (CDN).
